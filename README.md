# Kards卡牌剪影猜猜猜 - AstrBot 插件

机器人发送一张去掉了费用、国籍、特效、词条等文字的卡牌中央形象图片，群友根据图片猜出卡牌名称。

## 功能特点

- 📷 **自动裁剪**：根据配置比例裁剪掉卡牌UI文字，只保留中央插画
- 🎲 **随机出题**：从您提供的卡牌图片库中随机抽取
- ⏱️ **计时猜牌**：每轮限时猜牌，超时公布答案
- 📊 **积分与排行榜**：猜对得分，并显示群内排名
- 🔧 **可配置**：可调整裁剪比例、每日轮数、超时时间等
